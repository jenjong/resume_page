Research Topics

Interpretable Neural Network

Modeling, Optimization

The study of interpretable neural network includes is devleoping 1) a general neural network model with constraints, 2) an efficient estimation method for the constrained model, 3) statistical test of the complex output from the neural network. We expect that we can get a useful insight as well as powerful prediction performances from the developed model. We proposed the estimation method for multiple quantiles on the deep neural network.

With Sangjun

Dynamic Model

Modeling, Optimization, Regularization

We consider the non-stationary model of which the model parameters depend on the time. For example, the ability of a baseball team changes during the play season, and the patterns of extreme annual maximum precipitation varies in a long term. We model the dynamic patterns by time varying parameters and develope a efficient estimation method for the model.

Ranking model

Modeling, Asymptotics

This study aims to develop a general probability model for ranking procedure. We consider the Thurstond model and the Bradley-Terry model as a special case, and investigate the method for estimating the model parameters. We found that there are model misspecification problem which possibly misleads the estimated ranking. The ranking model can be extended to the recommendation system.
